# API-Monolitico_CP303061X
API REST monolítica para gerenciamento de usuários, desenvolvida com Node.js, Express e SQLite3, conforme requisitos da atividade prática.

A aplicação permite operações completas de CRUD com desativação lógica de usuários.

# Instruções do Projeto

1. Clonar o repositório:
    `git clone https://github.com/Rafaela-Mello/API-Monolitico_CP303061X.git`
    `cd api-usuarios`

2. Instalar as dependências
    `npm install`

3. Executando a Aplicação
    `node src/app.js`

4. A API será iniciada em:
    http://localhost:3000

5. Utilize o ThunderClient (extensão do VS Code) ou qualquer cliente HTTP (Postman, Insomnia) para testar os endpoints.
